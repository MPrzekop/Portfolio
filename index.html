Shaders

Gradient Based Stylized Shader

Shader inspired by Gooch shading model. Allows to define gradients which are used to remap lighting values, such as diffuse, specular and fresnell.

Useful for creating non-photorealistic, or even psychodelic shading.

Can be downloaded using Unity package manager. Created for Unity built-in pipeline.

Repository

Snow Shader

Basic snow shader adding glitter and subsurface scattering to a standard shader.

Main logic is inside ASE function, so the glitter and SSS can be used separately with ASE.

Can be downloaded using Unity package manager. Created for Unity built-in pipeline.

Repository

Tools

Node based texture generator/editor

Based on XNode and inspired by ASE texture editor enabling user to quickly modify or generate textures.

Created as a tool for repacking maps and editing noise textures, but can be used for vast number of usecases.

Created for Unity 2021.3, due to compatibility package is set for editor only, but if project uses .NET 4.x it should work also in runtime.

Repository

Unity inspector graph drawer

Package allows for drawing graphs in the inspector. Created for real time visualisation of interpolation values.

Useful also for debugging/analysing and drawing change over time.

Can be downloaded using Unity package manager.

Repository

Scriptable render pipeline study

Created from grounds-up rendering pipeline to explore what can be achieved with Unity's SRP.

Current pipeline provides implementation of Disney BSDF, 4 types of lights and SSR that doubles as SSGI.

Created with Unity 2020.3 and Core RP Library 10.9.0

Repository

Post Processes

Voronoi based mosaic post process

Bare-bones project created for a lecture I gave for the GUT science club Vertex PG. Still may be used as a base for something bigger.

Can achieve an interesting painterly-like effect.

Does not use PPS V2, uses Blit functionality. Compatible with Unity built-in pipeline.
